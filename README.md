What This Program Does
•	Manage books (add, remove, search)
•	Manage users (register, track borrowing)
•	Borrow and return books
•	Calculate fines for overdue books
•	Search by title, author, genre
•	Show library statistics
Menu Options
1.	Add new book
2.	Show all books
3.	Show available books
4.	Search by title
5.	Search by author
6.	Register new user
7.	Checkout book
8.	Return book
9.	Show library stats
10.	Run demo ← Try this first!
11.	Exit
Classes
Book - Represents a book
•	Properties: ISBN, title, author, genre, year
•	Can be borrowed and returned
•	Tracks overdue status
User - Represents a library member
•	Properties: ID, name, email, phone
•	Tracks borrowed books
•	Manages fines
Library - Manages the system
•	Stores books and users
•	Handles borrowing/returning
•	Provides search functions
